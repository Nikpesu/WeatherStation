🌦️ WeatherStation Web Installer

Flash, update and debug your device from the browser — no IDE, no drivers.

1Install / update firmware

Plug the ESP into a USB port, click Install, and pick the serial port. The right build is chosen automatically for your chip. This writes the firmware and the web UI.

⚠ Your browser can't do serial installs — use desktop Chrome or Edge. ⚠ Open this page over https (or localhost).
Installing does a full erase, so it also clears saved settings on the device. To keep your settings, update from the device's own page (⬆ Firmware Update) or with the per-file uploader instead.

🔧 Custom secrets.h

Bake your Wi-Fi/MQTT defaults + enabled sensors into the firmware at build time. Fill the fields (they update the file live), tweak the text if you like, then Download it into the project root and build from source.

ℹ️ The one-click Install above flashes a prebuilt binary and can't embed these — for a flashed device, just set everything in its Wi-Fi setup portal after install. This generator is for people building from source (pio run).

3Open a device on the network

Already flashed and on Wi-Fi? Enter its IP or hostname.local to open its web page (which has a live log under Current Config).

Opened in a new tab. A browser can't stream the device's log into this page — an https page isn't allowed to read an http device (mixed content) — so the device's own page shows the log instead.
